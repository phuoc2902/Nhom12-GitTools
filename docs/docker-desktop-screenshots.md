# Docker Desktop Screenshot Guide

Để đạt yêu cầu báo cáo, bạn cần chụp ít nhất 3 màn hình trong Docker Desktop:

1. **Images**
   - Hiển thị image `phuoc4/truonghuyphuoc:v1.0`.
   - Ghi chú: show tag `v1.0`, kích thước image, và trạng thái image.

2. **Containers / Apps**
   - Hiển thị container `nhom12-gittools-web` đang chạy.
   - Ghi chú: show cổng `8080`, tên container, trạng thái `Up`.

3. **Docker Hub / Registry**
   - Nếu bạn đã đăng nhập Docker Desktop với Docker Hub, chụp màn hình phần `Docker Hub` hoặc `Registry`.
   - Ghi chú: show tài khoản `phuoc4` hoặc image `phuoc4/truonghuyphuoc:v1.0`.

## Ghi chú khi chụp hình

- Chụp rõ phần `tag` và `repository`.
- Nếu không có Docker Hub trên Docker Desktop, chụp màn hình `Images` đủ.
- Có thể thêm chú thích nhỏ bên cạnh ảnh để giải thích mỗi màn hình.
